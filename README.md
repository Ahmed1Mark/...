## الوصف العام

البوت هو بوت متعدد الوظائف تم تطويره باستخدام JavaScript وإصدار Discord.js v13. يعمل البوت على إصدار Node.js 10 ويقدم مجموعة من الأنظمة والإمكانيات التي تسهل إدارة السيرفر وتفاعل الأعضاء.
